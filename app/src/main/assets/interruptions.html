Interruptions

Interruptions can be added using the buttons Add Interruptions in the Innings 1 and Innings 2. For each interruption, you need to give:

  • The number of overs played before the interruption occurred
    • For the first interruption, this is the number of overs played between the start of the innings and the start of this interruption
    • For subsequent interruptions, this is the number of overs played between the end of the previous interruption and the start of this interruption

  • The number of wickets lost by the team at this point in the match
    • This cannot be fewer than the number of wickets lost at an earlier interruption!

  • The number of overs play lost due to this interruption
    • The total of the numbers of overs listed in the 'Overs played' and 'Overs lost' columns cannot exceed the number of overs avaliable to play by considering antecedent interruption if not, number of overs in the innings.

On Click of done, it will take to innings page for futher calculation
